# 구독형 이커머스 멤버십 고객 이탈 예측 & RFM 세분화

> **개인팀(황채원)** ·
> 빅데이터/AI 프로젝트 — *데이터로 이탈을 예측하고, 떠나기 전에 붙잡는다.*

---

## 한 줄 요약
구독 비즈니스의 핵심 KPI인 **고객 이탈(Churn)** 을 머신러닝으로 예측하고,
**RFM 군집분석**으로 고객을 세분화하여 **"고가치 × 고위험" 고객을 우선 타깃하는 선제 리텐션 체계**를 제안한 프로젝트.

## 왜 이 주제인가 (비즈니스 가치)
- 신규 고객 획득 비용은 기존 고객 유지 비용의 약 5배 — 구독 모델에서 이탈 방어는 곧 수익이다.
- 이탈을 "사후 집계"가 아니라 **"사전 예측"** 하면, 떠나기 전에 개입할 수 있다.
- 이탈확률 점수 + 고객 가치 세그먼트를 결합해 **마케팅 예산을 ROI 높은 곳에 집중**한다.

## 데이터
- **9,000명 / 20개 변수**의 합성(synthetic) 고객 데이터를 직접 생성
- 실데이터를 쓰지 않아 **개인정보·저작권 이슈가 전혀 없음**, 동시에 실제 구독 커머스의
  행동 패턴(최근성·구매빈도·만족도·자동결제 등)을 인과관계까지 반영해 모사
- 이탈률 20.2%의 현실적인 **불균형 데이터**

## 핵심 결과
- **최종 모델: 로지스틱회귀** (ROC-AUC 0.813, 이탈 재현율 0.73) — class_weight로 불균형 대응
- 핵심 이탈 요인: **최근주문경과일 · 자동결제 해지 · 로그인/주문 빈도 · 만족도 · 가입개월수**
- **이탈 고위험 고객 약 2,200명**을 사전 식별하는 스코어링 산출물 생성
- RFM 4개 세그먼트(VIP / 성장잠재 / 일반 / 휴면위험)별 차별화 마케팅 전략 제시
